# Tabs

## Installation

Using the react components:

    import TabGroup from '@vividseats/vivid-design-patterns/dist/molecules/TabGroup';
    import Tab from '@vividseats/vivid-design-patterns/dist/atoms/Tab';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/tabs.scss';

## Basic Usage

## Add custom event listener callback functions

<Playground>
    <TabGroup>
        <Tab
            onClick={function(e) {
                e.preventDefault();
                alert('Called onClick');
            }}>
            Tab with onClick callback
        </Tab>
        <Tab
            onMouseEnter={function(e) {
                alert('Called onMouseEnter');
            }}>
            Tab with onMouseEnter callback
        </Tab>
        <Tab
            onMouseLeave={function(e) {
                alert('Called onMouseLeave');
            }}>
            Tab with onMouseLeave callback
        </Tab>
    </TabGroup>
</Playground>
