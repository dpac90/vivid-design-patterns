# Modal
## Installation
Using the react components:

    import Modal from '@vividseats/vivid-design-patterns/dist/molecules/Modal';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/_modals.scss';

## Basic Example
`
React: className="vdp-modal" dataState="opened"
HTML: class="vdp-modal" data-state="opened"
`

## Props Example
`dataState="opened" title="Modal Title" type="sheet" onClose={() => window.alert('on close fired')`
<Playground style={ { minHeight: '400px', height: '100%' } } >
    <Modal dataState="opened" title="Modal Title" type="sheet" onClose={() => window.alert('on close fired')}>
        This sheet modal is uses a prop for the modal title, passes in the body text, and uses the default modal footer and dismiss button.
    </Modal>
</Playground>

## Sub-Component Example
```
<Modal>
    <Modal.Header />
    <Modal.Body />
    <Modal.Footer />
</Modal>
```
<Playground style={ { minHeight: '400px', height: '100%' } } >
    <Modal dataState="opened">
        <Modal.Header><Headline importance={6}>Custom Modal H4 Header</Headline></Modal.Header>
        <Modal.Body>
            <p>This modal is using subcomponents for the modal header, body, and footer.</p>
        </Modal.Body>
        <Modal.Footer>
            <Button importance="tertiary" onClick={() => window.alert('custom button clicked')}>Custom button</Button>
        </Modal.Footer>
    </Modal>
</Playground>

## Full Screen Example
```
    <Modal type="full-screen"/>
```
The full screen modal is available only for mobile screen sizes and doesn't have a default modal footer subcomponent.
