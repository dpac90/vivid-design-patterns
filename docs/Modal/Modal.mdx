# Modal
## Installation
Using the react components:

`import Modal from @vividseats/vivid-design-patterns/dist/molecules/Modal;`

Using the scss files:

`@import @vividseats/vivid-design-patterns/src/styles/_modals.scss;`

## Basic Example
`
React: className="vdp-modal" dataState="opened"
HTML: class="vdp-modal" data-state="opened"
`

## Props Example
`dataState="opened" title="Modal Title" type="sheet" onClose={() => window.alert('on close fired')`

## Sub-Component Example
```
<Modal>
    <Modal.Header />
    <Modal.Body />
    <Modal.Footer />
</Modal>
```
<Playground style={ { minHeight: '400px', height: '100%' } } >

        <Modal.Header importance={4}>Custom Modal H4 Header</Modal.Header>
        <Modal.Body>
            <p>This modal is using subcomponents for the modal header, body, and footer.</p>
        </Modal.Body>
        <Modal.Footer>
            <Button importance="tertiary" onClick={() => window.alert('custom button clicked')}>Custom button</Button>
